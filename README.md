# CSU Sénégal - Vérification Carte Bénéficiaire

Application web pour vérifier la validité des cartes de bénéficiaires de la Couverture Sanitaire Universelle du Sénégal.

## Fonctionnalités
- Vérification de carte par QR code
- Affichage des informations du bénéficiaire
- Validation de la période de couverture
- Interface professionnelle et responsive

## Technologies
- Angular 15+
- Bootstrap 5
- TypeScript
- HTML5/CSS3

## Installation locale
```bash
npm install
ng serve
